- 一种应用
- 即**万维网（World Wide Web）**
- **按需**操作

### Web页面
- Web page，也叫文档
- 由**对象**组成
- 一个**对象（object）**只是一个文件。如：HTML文件、jpg图片、java小程序、视频
- 每个对象可以通过[[#URL]]地址寻址
- 一个Web页面一般由一个HTML基本文件（base HTML file）和几个引用对象（通过URL）组成

### Web浏览器
- 实现[[HTTP]]的客户端
- 例如：Chrome、FireFox
- 更多细节见[[现代浏览器内部揭秘]]
### Web服务器
- 存储Web对象，通过[[#URL]]寻址
- 实现[[HTTP]]的服务器端
- 例如：Apache

### URL
- 两部分：存储对象的服务器主机名、对象的路径名
